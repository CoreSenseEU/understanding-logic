## Types
### **concept** `$tType` { #concept data-toc-label='concept' }
An instance of a class of [phenomenon](phenomenon.md) in the agent's world model. 

A [concept](#concept) is a [property](property.md) of a [modelet](modelet.md).

!!! example
    - chair
    - dog
    - the color pink
    - sunrise
